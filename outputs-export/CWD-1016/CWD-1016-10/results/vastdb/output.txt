

## Root Cause

The vulnerability is in the `gaussian_blur` function at bad.c:460. A `memcpy(dst, src, w)` is called where `w` is the image width from `plane->width`. The function uses `w` as the byte count without validating it against the actual buffer linesizes (`dst_linesize` and `src_linesize`). If `w` exceeds the allocated buffer linesize, the memcpy writes beyond the buffer bounds, causing an out-of-bounds write (CWE-787).
